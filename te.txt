# Tutorial Completo: Transformando o Zorin OS no Visual DeadSec de Watch Dogs 2

Este guia detalhado transforma seu Zorin OS na estética hacker do grupo DeadSec do jogo Watch Dogs 2, com um visual cyberpunk completo incluindo cores neon, fontes digitais e elementos de interface futuristas.

## 1️⃣ Preparação Inicial

Primeiro, vamos instalar as ferramentas necessárias:

```bash
# Instalar ferramentas essenciais
sudo apt update
sudo apt install git curl wget unzip gnome-tweaks dconf-editor gconf-editor plank conky -y

# Ferramentas de personalização adicionais
sudo apt install lxappearance nitrogen picom rofi neofetch cmatrix -y
```

## 2️⃣ Tema e Paleta de Cores DeadSec

### Instalação de Temas GTK

```bash
# Criar pastas de temas se não existirem
mkdir -p ~/.themes ~/.icons ~/.fonts

# Instalar tema Dracula (base escura)
git clone https://github.com/dracula/gtk.git ~/.themes/Dracula

# Instalar tema Cyberpunk Neon
git clone https://github.com/Roboron3042/Cyberpunk-Neon.git  
cd Cyberpunk-Neon  
mkdir -p ~/.themes/Cyberpunk-Neon  
cp -r theme/* ~/.themes/Cyberpunk-Neon/  
cd .. && rm -rf Cyberpunk-Neon  

# OU use outro tema neon (alternativa)  
git clone https://github.com/daniruiz/flat-remix-gtk.git  
cd flat-remix-gtk  
cp -r Flat-Remix-GTK-Green-Dark ~/.themes/  
cd .. && rm -rf flat-remix-gtk 
```

### Instalação de Ícones

```bash
# Instalar pacote de ícones Papirus (base)
sudo add-apt-repository ppa:papirus/papirus
sudo apt update && sudo apt install papirus-icon-theme -y

# Instalar versão neon dos ícones
wget https://github.com/Adapta-Projects/Papirus-Adapta/releases/download/20191022/Papirus-Adapta-Nokto.tar.xz
tar -xf Papirus-Adapta-Nokto.tar.xz -C ~/.icons/
rm Papirus-Adapta-Nokto.tar.xz

# Cursor hacker
git clone https://github.com/keeferrourke/capitaine-cursors.git
cd capitaine-cursors
./build.sh
mkdir -p ~/.icons/capitaine-cursors
cp -r dist/ ~/.icons/capitaine-cursors/
cd .. && rm -rf capitaine-cursors
```

### Configuração do Tema

Abra o aplicativo "Aparência" do Zorin OS:

1. Vá em "Temas" e selecione "Dracula" como tema Global
2. Vá em "Ícones" e selecione "Papirus-Adapta-Nokto"
3. Vá em "Cursores" e selecione "capitaine-cursors"

## 3️⃣ Fontes Estilo DeadSec

```bash
# Baixar fontes estilo hacker
mkdir -p ~/fonts-temp
cd ~/fonts-temp

# Fonte principal para interface - similar à usada pelo DeadSec
wget https://github.com/ryanoasis/nerd-fonts/releases/download/v2.1.0/Hack.zip
unzip Hack.zip -d ~/.fonts/

# Fonte para terminal
wget https://github.com/ryanoasis/nerd-fonts/releases/download/v2.1.0/FiraCode.zip
unzip FiraCode.zip -d ~/.fonts/

# Fonte para textos estilizados do DeadSec
wget https://github.com/ryanoasis/nerd-fonts/releases/download/v2.1.0/SourceCodePro.zip
unzip SourceCodePro.zip -d ~/.fonts/

# Atualizar cache de fontes
fc-cache -fv
cd && rm -rf ~/fonts-temp
```

### Configurar Fontes no Sistema

```bash
# Configurar fontes para o sistema
gsettings set org.gnome.desktop.interface font-name 'Hack Nerd Font 10'
gsettings set org.gnome.desktop.interface document-font-name 'Hack Nerd Font 10'
gsettings set org.gnome.desktop.interface monospace-font-name 'FiraCode Nerd Font Mono 10'
gsettings set org.gnome.desktop.wm.preferences titlebar-font 'Hack Nerd Font Bold 10'
```

## 4️⃣ Dock e Barra de Tarefas Estilo Hacker

### Configurar Plank Dock

```bash
# Instalar tema neon para o Plank
mkdir -p ~/.local/share/plank/themes/
git clone https://github.com/dikiaap/frost-plank-theme.git
cd frost-plank-theme
cp -r Frost-Plank-Theme/ ~/.local/share/plank/themes/
cd .. && rm -rf frost-plank-theme

# Configurar Plank
dconf write /net/launchpad/plank/docks/dock1/theme "'Frost-Plank-Theme'"
dconf write /net/launchpad/plank/docks/dock1/zoom-enabled true
dconf write /net/launchpad/plank/docks/dock1/zoom-percent 150
dconf write /net/launchpad/plank/docks/dock1/icon-size 48
dconf write /net/launchpad/plank/docks/dock1/position "'bottom'"
```

Para iniciar o Plank automaticamente:
1. Abra "Configurações" > "Inicialização"
2. Adicione novo aplicativo > selecione "Plank"

### Personalizar Barra Superior

Para Zorin OS com GNOME:

```bash
# Instalar extensões GNOME
sudo apt install gnome-shell-extensions chrome-gnome-shell -y

# Adicionar customização à barra superior 
gsettings set org.gnome.desktop.interface show-battery-percentage true
gsettings set org.gnome.desktop.interface clock-show-seconds true
gsettings set org.gnome.desktop.interface clock-show-date true
```

Visite https://extensions.gnome.org e instale:
- "User Themes"
- "Dash to Panel" ou "Dash to Dock"
- "Dynamic Panel Transparency"
- "Clipboard Indicator"
- "NetSpeed"

## 5️⃣ Terminal Customizado Estilo DeadSec

### Configuração do Terminal

```bash
# Instalar ZSH
sudo apt install zsh -y

# Instalar Oh My Zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# Instalar PowerLevel10k (tema de terminal hacker)
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/themes/powerlevel10k

# Instalar plugins úteis
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
```

Edite o arquivo ~/.zshrc:
```bash
nano ~/.zshrc
```

Modifique para:
```
ZSH_THEME="powerlevel10k/powerlevel10k"
plugins=(git zsh-autosuggestions zsh-syntax-highlighting)
```

Configurações do perfil do terminal:
1. Abra o Terminal e vá para "Preferências"
2. Crie um novo perfil "DeadSec"
3. Configure:
   - Fundo: Transparência 15%
   - Cor de fundo: #121212
   - Cor de texto: #00FF41 (verde neon)
   - Fonte: FiraCode Nerd Font Mono 10
